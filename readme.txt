Live Forex Table plugin displays unlimited foreign exchange rate in tabular format based on Google Finance.  

== Description ==

Live Forex Table plugin displays unlimited foreign exchange rate in tabular format based on Google Finance. You can choose your currency and which you want to exchange with. It is very simple tool.You can just use [live_forex_rate] short code where you want to display the table. This plugin allows to create unlimited foreign exchange list like in news paper / magazine. 


== Installation ==

1. Download, unzip and upload to your WordPress plugins directory
2. Activate the plugin within you WordPress Administration
3. Add currencies from settings -> Live Forex Settings page and use [live_forex_rate]  short code 

== Frequently Asked Questions ==
= What is base currency? =
It is the currency in which other currencies are converted to. Generally, user use their own currency as the base currency.
= What is other currency? =
It is the currency which you want to convert with your base currency.
